# 🌸 GalGame Web Chat (GWC) - 沉浸式 AI 虚拟伴侣系统

这是一个基于 React + Vite + Tailwind CSS 构建的 Web 端全沉浸式视觉小说（GalGame）风格 AI 聊天应用。它不仅是一个聊天窗口，更是一个集成了 **Live2D 动态同步**、**长期记忆流**、**隐式日程管家**机制的赛博私人助理。

项目完全运行在浏览器纯前端环境中，数据采用 IndexedDB 本地存储，绝对保障您的隐私安全。

## ✨ 核心特性 (Features)

### 🎭 视听交互引擎
* **Live2D 看板娘支持**：支持本地无缝导入多套 `.model3.json` Live2D 模型，自动记忆每套模型的独立缩放与位置参数。
* **沉浸式视听空间**：支持多背景图切换、本地 BGM 列表循环/随机播放、打字机流式输出、长文本平滑滚轮翻页与一键 SKIP。
* **同声传译双语模式**：支持 AI 同时生成用于 TTS 的源语言与用于屏幕显示的母语（翻译版）。

### 🧠 记忆与灵魂架构
* **记忆增强与上下文压缩**：内置自动归档机制，当对话达到指定轮数（如150轮），后台静默唤醒大模型进行记忆提炼，无缝替换上下文，彻底解决长对话遗忘和 Token 爆炸问题。
* **自然语言日程管家**：直接对 AI 说“X分钟后提醒我喝水”，系统会隐式解析你的意图并在后台上闹钟，时间一到 AI 会主动出声提醒。（暂未完善，需手动添加日程）

### 💾 坚固的本地数据系统
* **纯本地 IndexedDB 存储**：所有聊天记录、角色卡设定、背景音乐、模型文件全量储存在您的浏览器本地，不经过任何第三方数据库。
* **AUTO 自动存档保护**：后台每 5 分钟（可调）静默更新自动存档。支持 START 覆盖保护与存档插槽迁移。
* **一键全量备份**：支持将所有存档、设置、图片、音乐、角色卡打包为 `.json` 文件导出，方便跨设备迁移。

## 🚀 快速开始 (Quick Start)
<img width="2458" height="1471" alt="屏幕截图 2026-03-22 105045" src="https://github.com/user-attachments/assets/fd64492b-f892-41a0-82ef-6933616e0e61" />

本项目环境依赖于 **Node.js**。如果您电脑上还未安装，请先前往 [Node.js 官网](https://nodejs.org/) 下载并安装。

### 选项 A：小白一键启动版（Windows 推荐）
1. 克隆或下载本仓库代码到本地。
2. 双击根目录下的 `启动.bat` 文件。
3. 脚本会自动为您安装所需依赖并启动本地服务器，随后在浏览器中打开聊天界面。<br>
4.[注意]TTS需要本地部署GPT-SoVITS作为后端或使用云端服务。<br>
5.初次使用请先前往设置配置大模型接口API，本项目兼容OpenAI格式的API接口。<br>
